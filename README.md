# Panda Digital Mini-Dashboard

## Display wiring

| Dsiplay | Raspberry Pi 4 |
|---------|----------------|
| VCC     | 3,3V (1)       |
| GND     | GND (6)        |
| CS      | GPIO 8 (24)    |
| RESET   | GPIO 27 (13)   |
| A0      | GPIO 25 (22)   |
| SDA     | GPIO 10 (19)   |
| SCK     | GPIO 11 (23)   |
| LED     | GPIO 18 (12)   |

## Button Wiring

| Button | Raspberry Pi 4 |
|--------|----------------|
| VCC    | 3,3V (17)      |
| PIN    | GPIO 15 (10)   |
